# 📘 Blacktokki Notebook 사용자 가이드

**Blacktokki Notebook**은 마크다운 기반의 **지식 관리 및 시간 관리 도구**입니다. 사용자는 다양한 주제의 노트를 자유롭게 작성하고 구조화하며, 일정과 연결하여 추적할 수 있습니다. 모바일과 웹 환경에서 모두 사용할 수 있으며, **자기 주도 학습**, **업무 관리**, **지식 아카이빙**, **회고 기록** 등에 유용하게 활용할 수 있습니다.

## 주요 특징

* ✅ 마크다운 에디터를 통해 자유롭게 노트를 작성할 수 있습니다
* 📁 폴더와 하위 노트 구조로 내용을 계층적으로 정리할 수 있습니다
* 🧭 문단 제목을 기반으로 자동 생성되는 트리 뷰로 빠르게 탐색할 수 있습니다
* 🔍 제목과 문단 이름, 링크, 키워드까지 포함한 고급 검색 기능을 제공합니다
* 🔁 특정 문단을 다른 노트로 이동할 수 있습니다
* 📆 일정 정보를 자동으로 감지하여 타이머 형태로 시각화할 수 있습니다
* 🧠 자동완성 기능으로 링크와 자식 노트를 빠르게 작성할 수 있습니다
* 🧳 Markdown 형식으로 노트를 내보내거나 다시 가져올 수 있습니다

## 시작하기 (Getting Started)

### 🔍 노트 검색 및 생성

* 상단 검색창에서 키워드 입력 시 기존 노트 탐색 및 새 노트 생성 가능
* 새로운 제목 입력 후 `Enter` 키를 누르면 자동으로 해당 노트 생성 및 진입

### ✏️ 노트 편집

* 마크다운 기반 에디터 제공
* 자동완성 트리거:
  * `[` → 내부 노트 링크. `[키워드` 입력 시 다른 노트 제목뿐만 아니라 **현재 노트의 문단 제목** 또는 **새로운 하위 노트(예: `현재노트/키워드`)** 생성을 제안합니다.
  * `http` → 외부 링크 미리보기

## 화면별 기본 사용법

### 🏠 홈

* 첫 진입 시 가장 먼저 보이는 메인 화면입니다
* 최근 열람한 노트 목록, 편집 제안, 타임라인 기능에 빠르게 접근할 수 있습니다
* 모바일 환경에서는 하단 탭으로 주요 메뉴에 접근하며, 웹에서는 좌측 메뉴(Drawer)를 통해 모든 기능에 빠르게 이동할 수 있습니다

### 📄 노트 페이지

* 노트의 제목과 본문, 그리고 구조화된 문단 목록이 함께 제공됩니다
* 사이드에 표시되는 문단 목록은 노트 내 제목(H1~H6)을 기반으로 자동 생성되며, 클릭 시 해당 위치로 이동합니다
* 상단 버튼을 통해 노트를 편집하거나, 문단 이동 또는 편집 이력을 확인할 수 있습니다

### ✏️ 노트 편집

* 위지윅/마크다운 형식으로 작성 가능한 에디터 화면입니다
* 노트의 본문을 자유롭게 작성하거나 수정할 수 있으며, 자동 저장 기능과 수동 저장 기능이 모두 제공됩니다
* 내부 노트 링크, 외부 링크 자동완성 기능을 활용할 수 있습니다

### 🗂 최근 노트

* 사용자가 가장 최근에 열람한 노트를 시간순으로 나열하여 보여줍니다
* 클릭 시 해당 노트로 바로 이동하며, 원하지 않는 노트는 리스트에서 제거할 수 있습니다

### 🗂 보드

* 노트의 문단들을 칸반/스크럼 보드 형태로 시각화하고 관리할 수 있는 기능입니다.
* 사용자는 '보드'를 생성, 수정, 삭제할 수 있습니다.
* 각 보드의 '컬럼(Column)'은 사용자가 지정한 **개별 노트**로 구성됩니다. (예: 'To Do' 노트, 'In Progress' 노트)
* 각 컬럼의 '카드(Card)'는 해당 노트 내의 **특정 레벨 제목(예: H2, H3)을 가진 문단**들로 자동 구성됩니다. (보드 설정에서 헤더 레벨 지정 가능)
* **스크럼 보드**에서는 카드 기준 헤더의 상위 레벨(Header Level - 1)을 기준으로 **행(Row)**이 자동 생성되어, 프로젝트의 단계나 담당자별로 카드를 더 입체적으로 구조화할 수 있습니다.
* 카드를 드래그 앤 드롭하여 다른 컬럼으로 이동시킬 수 있습니다.
* 카드를 다른 컬럼으로 이동시키면, 해당 문단의 내용이 실제로 다른 노트로 이동됩니다. (예: 'To Do' 노트의 H3 문단을 'In Progress' 노트로 이동)

### 🗃 아카이브

* 저장된 노트의 이전 버전(스냅샷)을 확인할 수 있는 화면입니다
* 노트 제목별로 과거 저장 시점의 내용을 시간순으로 비교할 수 있습니다
* 버전 관리와 과거 내용 복원이 필요할 때 유용하게 사용됩니다

## 고급 기능

### 📝 듀얼 에디터 지원

* 노트 편집 화면은 `M↓` 버튼으로 WYSIWYG 방식과 마크다운 방식 두 가지 모두를 자유롭게 변환할 수 있습니다.
* 두 편집 방식은 실시간으로 동기화되어 한쪽을 수정하면 다른 쪽에서도 바로 반영됩니다
* 사용자는 마크다운에 익숙하든, 시각적인 편집을 선호하든 모두 편리하게 작성할 수 있습니다

### 📑 문단 구조 자동 인식

* 노트에 작성된 제목은 자동으로 문단 구조로 변환되어 트리 형태로 표시됩니다
* 사용자는 이 트리를 통해 특정 문단으로 바로 이동하거나 전체 구조를 시각적으로 파악할 수 있습니다

### ⌞⌝ 문단 내용 전체 보기 기능
* 특정 문단으로 이동했을 때, 상단의 '펼치기' / '접기' 아이콘을 통해 해당 문단의 내용만 볼지, 하위 문단들의 내용까지 포함해서 볼지 토글할 수 있습니다.
* 하위 문단을 포함해서 보는 경우(펼치기), 해당 문단의 제목부터 모든 하위 문단의 내용이 하나의 노트처럼 표시됩니다.

### 🔗 링크 인식 및 연결

* 노트 내 삽입된 링크는 자동으로 인식되어 다른 노트 또는 외부 웹 페이지로 연결됩니다
* 다른 노트로 연결되는 링크는 검색이나 자동완성 결과에서 추천 항목으로 표시됩니다

### ⏱ 노트별 일정 자동 감지

* 타임라인에서 지원하는 날짜 형식으로 각 노트에서 작성된 일정을 자동으로 표시합니다
* 각 노트별 현재 일정은 타이머 바 형태로 시각적으로 표현되어 현재 상태를 쉽게 파악할 수 있습니다

### 🔍 검색 기록과 자동완성

* 검색한 키워드는 자동으로 저장되어 다음 검색 시 추천 항목으로 활용됩니다
* 최근 사용한 키워드를 빠르게 불러와 반복 입력을 줄일 수 있습니다

### 🔄 노트 및 문단 이동 (재배치)

* 전체 노트 또는 특정 문단을 다른 제목이나 위치로 이동하여 지식 구조를 재정비할 수 있습니다.
* **하위 노트 포함 이동**: 상위 노트의 이름이나 위치를 변경할 때, 해당 노트에 속한 모든 하위 노트를 한 번에 이동시킬 수 있습니다.
* **역링크 업데이트**: 이동 대상인 노트나 문단을 참조하고 있는 다른 노트들의 내부 링크(역링크)를 찾아 새 경로로 일괄 자동 업데이트하여 연결 끊김을 방지합니다.
* **변경사항 미리보기**: 이동 전 구조 변경 내용과 텍스트의 변경점(Diff)을 미리보기로 확인하여 안전하게 편집할 수 있으며, 이동하려는 대상 노트가 이미 존재할 경우 경고를 확인하고 덮어쓸 수 있습니다.

### 💾 저장 방식과 계정 연결

* 로컬 계정 또는 외부 계정을 선택하여 사용할 수 있으며, 계정에 따라 저장 위치가 달라집니다
* 온라인 상태에서는 자동 동기화가 활성화되어 최신 상태가 유지됩니다

### 🗃 스냅샷 기록 및 버전 비교

* 노트 저장 시 이전 버전이 자동 보존되며, 아카이브 화면에서 과거 내용과 비교할 수 있습니다

### 📌 탭 고정

* 현재 탭 또는 최근 변경사항에서 더블 클릭 또는 길게 클릭시 탭 목록으로 고정됩니다.
* X 버튼을 눌러 고정을 해제할 수 있습니다.

### ⇄ 탭 순서 변경

* 탭 목록에서, 항목을 위아래로 드래그하여 순서를 자유롭게 변경할 수 있습니다.
* 변경된 순서는 즉시 반영되며 자동으로 저장됩니다. 자주 사용하는 노트를 상단에 배치하여 작업 효율을 높일 수 있습니다.

### 🔐 프라이빗 모드

* 공개적인 장소에서 노트를 열람하거나 화면 공유 시, 민감한 정보를 숨길 수 있는 기능입니다.
* **숨김 처리 방법:** 노트 제목(또는 하위 경로의 제목)이 `.` (마침표)로 시작하면 해당 노트와 그 하위 노트는 자동으로 프라이버시 보호 대상이 됩니다.
    * 예시: `.비밀일기`, `프로젝트/.대외비`, `개인정보/.금융`
* **독립된 탭 목록:** 프라이빗 모드가 활성화되면 홈 화면의 '탭 목록'이 **'탭 목록 - 프라이빗 모드'**로 전환되며 일반 모드와 탭 기록이 완전히 분리됩니다.
* **보안 강화 (OTP 인증):** '프라이빗 모드 진입 시 OTP 요구' 옵션을 활성화하면, 모드에 진입하거나 해당 보안 설정을 해제할 때 OTP 인증 모달을 통한 추가 검증을 수행합니다. (온라인 계정 및 OTP 활성 상태에서 지원)
* **자동 해제 타이머:** '10분 미사용 시 자동 해제' 옵션을 켜두면, 마지막 활동 이후 10분간 조작이 없을 경우 프라이빗 모드가 자동으로 비활성화됩니다.
* 모드가 꺼진(Off) 상태에서 URL 등을 통해 숨겨진 노트에 직접 접근할 경우, 경고 문구가 표시되며 내용을 볼 수 없습니다.

### 🔐 개인 액세스 토큰(PAT) 관리

* 외부 서비스 및 애플리케이션 연동에 필요한 개인 액세스 토큰(PAT)을 직접 생성하고 관리할 수 있습니다.
* 계정 수정 모달 내 전용 섹션에서 새 토큰을 발급받을 수 있으며, 보안을 위해 발급 직후 단 한 번만 토큰 값이 노출되므로 즉시 복사하여 안전한 곳에 보관해야 합니다.
* 현재 활성화된 토큰 목록과 만료일을 확인하고, 더 이상 사용하지 않는 토큰은 언제든지 삭제하여 보안을 유지할 수 있습니다.

### ✨ MCP 서버 연동

* 발급한 개인 액세스 토큰으로 Blacktokki Notebook의 MCP 서버에 연동이 가능합니다.
* Claude Desktop 기준 MCP 설정방식은 다음과 같습니다:
```json
{
  "mcpServers": {
    "blacktokkiNotebook": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://blacktokki.kro.kr/agent/sse",
        "--header",
        "Authorization: Bearer ***"
     ]
    }
  }
}
