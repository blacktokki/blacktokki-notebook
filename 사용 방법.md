# 📘 Blacktokki Notebook 사용자 가이드

**Blacktokki Notebook**은 마크다운 기반의 **지식 관리 및 시간 관리 도구**입니다. 사용자는 다양한 주제의 노트를 자유롭게 작성하고 구조화하며, 일정과 연결하여 추적할 수 있습니다. 모바일과 웹 환경에서 모두 사용할 수 있으며, **자기 주도 학습**, **업무 관리**, **지식 아카이빙**, **회고 기록** 등에 유용하게 활용할 수 있습니다.

## 주요 특징

* ✅ 마크다운 에디터를 통해 자유롭게 노트를 작성할 수 있습니다
* 📁 폴더와 하위 노트 구조로 내용을 계층적으로 정리할 수 있습니다
* 🧭 문단 제목을 기반으로 자동 생성되는 트리 뷰로 빠르게 탐색할 수 있습니다
* 🔍 제목과 문단 이름, 링크, 키워드까지 포함한 고급 검색 기능을 제공합니다
* 🔁 특정 문단을 다른 노트로 이동할 수 있습니다
* 📆 일정 정보를 자동으로 감지하여 타이머 형태로 시각화할 수 있습니다
* 🧠 자동완성 기능으로 링크와 자식 노트를 빠르게 작성할 수 있습니다
* 🧳 Markdown 형식으로 노트를 내보내거나 다시 가져올 수 있습니다

## 시작하기 (Getting Started)

### 🔍 문서 검색 및 생성

* 상단 검색창에서 키워드 입력 시 기존 문서 탐색 및 새 문서 생성 가능
* 새로운 제목 입력 후 `Enter` 키를 누르면 자동으로 해당 문서 생성 및 진입

### ✏️ 문서 편집

* 마크다운 기반 에디터 제공
* 자동완성 트리거:

  * `[제목` → 내부 문서 링크
  * `http` → 외부 링크 미리보기

## 화면별 기본 사용법

### 🏠 홈

* 첫 진입 시 가장 먼저 보이는 메인 화면입니다
* 최근 열람한 노트 목록, 퍈집 제안, 타임라인 기능에 빠르게 접근할 수 있습니다
* 모바일 환경에서는 하단 탭으로 주요 메뉴에 접근하며, 웹에서는 좌측 메뉴(Drawer)를 통해 모든 기능에 빠르게 이동할 수 있습니다

### 📄 노트 페이지

* 노트의 제목과 본문, 그리고 구조화된 문단 목록이 함께 제공됩니다
* 사이드에 표시되는 문단 목록은 노트 내 제목(H1\~H6)을 기반으로 자동 생성되며, 클릭 시 해당 위치로 이동합니다
* 상단 버튼을 통해 노트를 편집하거나, 문단 이동 또는 편집 이력을 확인할 수 있습니다

### ✏️ 노트 편집

* 위즈웍/마크다운 형식으로 작성 가능한 에디터 화면입니다
* 문서의 본문을 자유롭게 작성하거나 수정할 수 있으며, 자동 저장 기능과 수동 저장 기능이 모두 제공됩니다
* 내부 노트 링크, 외부 링크 자동완성 기능을 활용할 수 있습니다

### 🗂 최근 노트

* 사용자가 가장 최근에 열람한 노트를 시간순으로 나열하여 보여줍니다
* 클릭 시 해당 노트로 바로 이동하며, 원하지 않는 노트는 리스트에서 제거할 수 있습니다

### 🧾 편집 제안

* 자동으로 감지된 오류나 문제가 있는 노트 목록을 표시합니다
* 감지된 문제는 다음과 같은 유형으로 분류되어 표시됩니다:

| 문제 유형                                | 설명                                                             |
| -------------------------------------- | ------------------------------------------------------------ |
| Empty paragraph                        | 문단의 제목은 있으나 본문이 비어 있는 경우                                     |
| Empty list                             | 문단 내 목록이 비어 있는 경우                                            |
| Duplicate paragraphs(...)              | 동일한 문단 제목이 반복된 경우                                            |
| Duplicate contents(...)                | 동일한 내용이 문단 내에서 반복된 경우                                       |
| Too high readability score: X > 3.0    | 읽기 난이도 점수가 기준치를 초과한 경우                                       |
| Unknown note link(...)                 | 존재하지 않는 노트로 연결된 링크가 있는 경우                                    |
| Empty parent note(...)                 | 상위 노트의 내용이 없는 경우                                             |
| Unlinked note keyword: ...             | 내용에 다른 노트로 연결 가능한 키워드는 있지만 해당 키워드에 링크가 연결되지 않은 경우            |
| Unlinked note keyword: ... => ...(...) | 내용에 다른 노트로 연결 가능한 키워드는 있지만 해당 키워드에 링크가 연결되지 않은 경우(다른 노트에서 작성된 링크 포함) |

* 문단이 비어있거나 링크가 잘못 연결된 경우, 또는 중복된 내용이 있는 경우 자동으로 분류됩니다
* 해당 항목을 클릭하면 문제 위치로 바로 이동하여 수정할 수 있습니다

### 📆 타임라인

* 날짜와 관련된 일정 기반 노트를 한눈에 확인할 수 있는 기능입니다
* 화면 상단의 날짜 선택 바 또는 달력에서 원하는 날짜를 선택하여 연결된 콘텐츠를 탐색할 수 있습니다
* 일정이 표시된 문단을 바로 확인하고 편집할 수 있습니다

### 🗃 아카이브

* 저장된 노트의 이전 버전(스냅샷)을 확인할 수 있는 화면입니다
* 노트 제목별로 과거 저장 시점의 내용을 시간순으로 비교할 수 있습니다
* 버전 관리와 과거 내용 복원이 필요할 때 유용하게 사용됩니다

## 고급 기능

### 📝 듀얼 에디터 지원

* 노트 편집 화면은 ✏️ 버튼으로 WYSIWYG 방식과 마크다운 방식 두 가지 모두를 자유롭게 변환할 수 있습니다.
* 두 편집 방식은 실시간으로 동기화되어 한쪽을 수정하면 다른 쪽에서도 바로 반영됩니다
* 사용자는 마크다운에 익숙하든, 시각적인 편집을 선호하든 모두 편리하게 작성할 수 있습니다

### 📑 문단 구조 자동 인식

* 노트에 작성된 제목은 자동으로 문단 구조로 변환되어 트리 형태로 표시됩니다
* 사용자는 이 트리를 통해 특정 문단으로 바로 이동하거나 전체 구조를 시각적으로 파악할 수 있습니다

### 🔗 링크 인식 및 연결

* 노트 내 삽입된 링크는 자동으로 인식되어 다른 노트 또는 외부 웹 페이지로 연결됩니다
* 다른 노트로 연결되는 링크는 검색이나 자동완성 결과에서 추천 항목으로 표시됩니다

### ⏱ 일정 자동 감지

* 날짜 형식으로 작성된 일정(예: 2024-06-01, 06/01\~06/05 등)을 자동으로 인식합니다
* 일정은 타임라인 화면과 타이머 바 형태로 시각적으로 표현되어 현재 상태를 쉽게 파악할 수 있습니다

### 🧠 노트 난이도 분석

* 각 노트의 문장 길이, 표현 방식 등을 분석하여 읽기 난이도를 평가합니다
* 너무 복잡하거나 긴 문장은 자동 감지되어 개선을 유도합니다

### 🔍 검색 기록과 자동완성

* 검색한 키워드는 자동으로 저장되어 다음 검색 시 추천 항목으로 활용됩니다
* 최근 사용한 키워드를 빠르게 불러와 반복 입력을 줄일 수 있습니다

### 🔄 문단 재배치 및 복사

* 노트의 특정 문단을 다른 노트로 이동하거나 복사할 수 있어 구조를 재정비할 수 있습니다
* 이동 전에는 미리보기로 구조 변경 내용을 확인할 수 있어 안전하게 편집 가능합니다

### 📦 백업 및 복원 기능

* 모든 노트는 Markdown 형식 또는 ZIP 파일로 백업할 수 있으며, 동일 형식으로 복원할 수 있습니다
* 장기적인 보관, 외부 공유, 로컬 저장 용도로 활용됩니다

### 💾 저장 방식과 계정 연결

* 로컬 계정 또는 외부 계정을 선택하여 사용할 수 있으며, 계정에 따라 저장 위치가 달라집니다
* 온라인 상태에서는 자동 동기화가 활성화되어 최신 상태가 유지됩니다

### 🗃 스냅샷 기록 및 버전 비교

* 노트 저장 시 이전 버전이 자동 보존되며, 아카이브 화면에서 과거 내용과 비교할 수 있습니다

### 🎯 랜덤 노트 열람

* 랜덤 버튼을 통해 무작위로 노트를 열람하며 새로운 관점을 얻을 수 있습니다

### 📌 열린 노트 고정

* 첫 열린 노트 또는 최근 변경사항에서 더블 클릭 또는 길게 클릭시 열린 노트 목록으로 고정됩니다.
* X 버튼을 눌러 고정을 해제할 수 있습니다.
